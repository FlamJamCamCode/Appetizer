## Altruism Country Deep-Dive Index

> **Status**: Growing index of country-level altruism appendices. Many numbers remain approximate (**\***); each country file is designed for iterative refinement as data is added.

### 1. Purpose

This index tracks:

- Which country deep dives exist.
- Rough classification (Western, Eastern, Middle Eastern, tax haven/financial hub, emerging, low-income).
- Pointers to the corresponding files under `countries/`.

### 3. Planned / Requested Categories

To approximate a “massive store” that covers:

- **Western countries**:
  - Examples: UK, France, Sweden, Canada, Australia.
- **Eastern countries**:
  - Examples: South Korea, China, Indonesia, Vietnam.
- **Middle Eastern countries**:
  - Examples: UAE, Qatar, Egypt, Jordan.
- **Tax havens / financial hubs**:
  - Examples: Monaco, Luxembourg, Switzerland, Cayman Islands.
- **Third world / low-income countries**:
  - Examples: Ethiopia, DR Congo, Afghanistan, Haiti.
- **Developing / emerging economies**:
  - Examples: Mexico, South Africa, Philippines, Bangladesh.

For each, the target structure mirrors existing deep dives:

- Macro context (population, GDP).
- Government social spending.
- ODA (given and/or received).
- Private philanthropy and informal altruism.
- Cause-area context using [Our World in Data – Causes of Death](https://ourworldindata.org/causes-of-death) and [Child mortality](https://ourworldindata.org/child-mortality).
- Cluster scenarios and NSS/Triad opportunities.

### 4. Notes on Data Quality

- **High-income countries** (US, Germany, Japan, Singapore):
  - Better fiscal and philanthropy data.
  - Detailed social spending and ODA figures.
- **Emerging / low-income countries** (India, Brazil, Nigeria, etc.):
  - Good macro and cause-of-death data from global sources.
  - Much weaker formal philanthropy and volunteering data.
  - Very large informal giving that is hard to quantify.

Each deep dive explicitly marks approximations with **\*** and should, over time, be updated with:

- Concrete data tables from:
  - OECD, World Bank, IMF.
  - National budgets and charity statistics.
  - CAF World Giving Index and similar surveys.

As more country files are created, add them here for navigation and cross-linking.
