## How It Works

The HTML file includes the guide `.js` files at load time (not runtime):

```html
<script src="world-peace.guide.js"></script>
<script src="freedom.guide.js"></script>
<script src="suffering.guide.js"></script>
<script src="wealth.guide.js"></script>
```

Each guide file defines a JavaScript constant:
- `GUIDE_WORLD_PEACE`
- `GUIDE_FREEDOM`
- `GUIDE_SUFFERING`
- `GUIDE_WEALTH`

The main script then assembles these into a `GUIDES` object and renders them dynamically.

## Adding New Guides

1. Create a new `your-guide-name.guide.js` file
2. Define `const GUIDE_YOUR_NAME = { ... }` with the guide structure
3. Include it in `guide-reader.html`: `<script src="your-guide-name.guide.js"></script>`
4. Add it to the `GUIDES` object: `yourName: GUIDE_YOUR_NAME`

## Card Colors

Guides can specify a `cardType` property to get custom styling:

**Available card types:**
- **`gold`** - Sacred/Featured guides (✦ Sacred ✦)
- **`white`** - Dream/Vision guides (◇ Dream ◇)
- **`green`** - Technical/NSS guides (🌿 NSS)
- **`black`** - Destruction/Zero guides (◆ (0) ◆)
- **`red`** - MWR 4/War guides (⚠ MWR 4 ⚠)
- **`pink`** - Sexual Liberation guides (♡ Liberation ♡)
- **`blue`** - Home/Sovereignty guides (🏠 Home)
- **Default** - Standard purple styling (if no cardType specified)

**Example:**
```javascript
const GUIDE_EXAMPLE = {
  "title": "My Guide",
  "description": "...",
  "cardType": "pink",  // ← Specifies pink card styling
  "tags": [...],
  "steps": [...]
};
```

## Guide Structure

Each guide contains:
```javascript
{
  "title": "The Question",
  "description": "Brief description",
  "cardType": "blue",  // Optional - defaults to purple if omitted
  "steps": [
    {
      "title": "Step Title",
      "content": [
        { "type": "paragraph", "text": "..." },
        { "type": "concept", "title": "...", "text": "..." },
        { "type": "example", "title": "...", "text": "..." },
        { "type": "key-insight", "title": "...", "text": "..." },
        { "type": "list", "items": [...] },
        { "type": "connection", "text": "..." }
      ]
    }
  ]
}
```
