# AR LLL Navigation Scene Prompt Template
## For Movie-Making AI

### Core Framework
Each scene description follows this structure:

---

## **SCENE [NUMBER]: [TITLE]**

### 1. **Physical Location**
- Where: [Specific place]
- When: [Time of day/year]
- Base Reality: [What exists without AR]

### 2. **AR Overlay State**
- Hue: [What color dominates - represents will flavor]
- Saturation: [Intensity level - represents probability/euphoria potential]
- Phase: [Which reality layer - goth/minimalist/fantasy/natural/custom]

### 3. **Character Will State**
- Seeking: [What character wants/needs]
- Current Emotion: [Emotional state]
- AR Guidance: [What the AR system shows them - colored paths, compatibility indicators, etc.]

### 4. **Movement/Action**
- Physical: [What character does in physical space]
- AR Response: [How AR overlay changes based on movement]
- Coalescence: [Moment of alignment/meeting/discovery]

### 5. **Sensory Details**
- Visual: [What camera shows - both physical and AR overlay]
- Audio: [Sounds, music, voice]
- Touch/Feel: [Tactile elements, haptics]
- Emotional Tone: [The feeling/vibe]

### 6. **Scene Resolution**
- Will Fulfillment: [Was character's seeking satisfied? Redirected? Deepened?]
- Next Guidance: [Where AR points them next]

---

## **EXAMPLE: "The Dual Rhythm Discovery"**
*Based on O2jam conversation - person discovering generative music through dual rhythm game*

### 1. **Physical Location**
- Where: Small studio apartment, cluttered desk with glowing monitor
- When: Late night, 2:47 AM
- Base Reality: Messy creative space, half-finished projects, dim blue LED light strips

### 2. **AR Overlay State**
- Hue: Deep purple transitioning to electric cyan (creativity-seeking + discovery-hunger)
- Saturation: LOW initially (aimless browsing), building to HIGH (creative flow imminent)
- Phase: Cyberpunk-Minimalist hybrid (person's preferred aesthetic)

### 3. **Character Will State**
- Seeking: Something new, non-repetitive, challenging both hands independently
- Current Emotion: Restless creative hunger, slightly bored with existing projects
- AR Guidance: Screen begins glowing with higher saturation purple around code editor; pathway forming toward "create rhythm game" impulse

### 4. **Movement/Action**
**Physical:** 
- Character's fingers tap desk unconsciously in two different rhythms (left: 4/4 kick pattern, right: melodic tapping)
- Leans forward, opens new project file
- Types "Make an O2jam clone"

**AR Response:**
- Purple path SURGES to bright violet-cyan
- Virtual notes begin falling in AR overlay around monitor
- Each keystroke shows color-coded rhythm visualization

**Coalescence:**
- Moment AI suggests dual independent lanes
- Character's eyes widen - both hands can play DIFFERENT music simultaneously
- AR overlay splits screen into left (percussion/purple-red) and right (melody/cyan-yellow)

### 5. **Sensory Details**
**Visual:** 
- Monitor glow reflects in character's eyes
- AR overlay shows two cascading note highways in split vision
- Code appears line by line with syntax highlighting matching AR colors
- Virtual drum/melody notes float around room

**Audio:**
- Typing sounds form rhythm
- First test: kick drum plays - character grins
- Then snare, hi-hat, bass building
- Right hand melody joins - **the moment of synthesis**
- Generative music system creates non-repeating patterns
- Sound builds like generative art ("electric sheep for music")

**Touch/Feel:**
- Fingertips tingle on keyboard
- Haptic feedback simulates note hits (in full AR version)
- Left hand feels bass/drums (heavy/sharp), right feels melody/harmony (light/flowing)

**Emotional Tone:**
- Starts: restless, seeking
- Middle: engaged, flow-state beginning  
- Peak: euphoria of discovery - "THIS is what I was looking for"
- Sustained: deep creative focus, time disappearing

### 6. **Scene Resolution**
**Will Fulfillment:** 
- Discovery satisfied ✓
- Creative hunger fed ✓  
- Challenge found (dual hand independence) ✓
- Non-repetitive generative music achieved ✓

**Next Guidance:**
- AR saturation remains HIGH (stay in this flow)
- New purple-cyan paths appear toward:
  - Add hold notes (sustained challenges)
  - Fix bugs (refinement path)
  - Share with others (social coalescence path)
- Character chooses refinement path - AR adjusts to show debugging opportunities glowing

## **Example: 3-Beat Quick Scene**

**Location:** City street corner | **AR Hue:** Orange-red (social hunger) | **Saturation:** Medium, rising

**Seeking:** Someone to jam music with

**Action:** Character walks past coffee shop → **AR Response:** Path glows bright orange toward doorway, saturation SPIKES → **Coalescence:** Inside, person with guitar has matching purple-orange AR glow (music + social compatibility)

**Feeling:** Loneliness → Anticipation → Connection spark

**Resolution:** Will redirecting (came for coffee, found jam partner) → **Next:** AR shows green path to nearby park (outdoor jam spot)

## **Usage for Movie AI**

Feed the AI:
1. The template structure above
2. Your narrative beats/story moments
3. Character will-states and seeking-patterns
4. Desired AR aesthetic (color schemes, phase realities)

The AI should generate:
- Visual scene descriptions with AR overlay specifications
- Character emotional/will progression
- How AR guidance creates narrative movement
- Coalescence moments (the dramatic beats)
- Sensory-rich details for cinematography

**Key Principle:** AR navigation makes the INVISIBLE visible - characters' inner seeking becomes external colored guidance, creating visual metaphor for will fulfillment journey.
