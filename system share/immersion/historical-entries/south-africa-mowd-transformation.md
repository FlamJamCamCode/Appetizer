# The South Africa MoWD Transformation: From "Kill the Boer" to Non-Lethal Territorial Competition

## Preface: MoWD Context

**MoWD (Metagame of World Dynamics)** = The recognition that ALL political systems are games.

**What MoWD reveals**:
- **Democracy** = Vote game (majority wins through numbers)
- **Capitalism** = Money game (wealth accumulation dominates)
- **Apartheid** = Racial hierarchy game (whites dominate through law)
- **Post-Apartheid SA** = Demographic game (blacks dominate through votes, threaten whites with violence)

**The pattern**: Every system declares "What is One/Equal" and allows "Might is Right" in the rest.

**This document**: Shows how South Africa negotiated from **"demographic domination + genocide threat"** (unacceptable to whites: extinction) to **"territorial separation + non-lethal competition"** (acceptable to both: space to build). This is ONE application of MoWD—finding a game where both groups can compete without genocide.

**The principle**: When current game threatens extinction ("Kill the Boer"), negotiate DIFFERENT game with acceptable losses (territorial competition, not racial extermination).

**The result**:

**The strategic implications**:
