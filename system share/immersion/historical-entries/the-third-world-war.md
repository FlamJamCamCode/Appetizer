## Preface: MoWD Context

**MoWD (Metagame of World Dynamics)** is not "non-lethal warfare"—it's the recognition that ALL political/conflict systems are games with rules.

**What MoWD reveals**:
- **All systems** declare "What is One/Equal" and allow "Might is Right" in the rest
- **Capitalism** = Money game (anyone can play, winner dominates)
- **Democracy** = Popularity game (votes equal, winner rules)
- **Traditional War** = Organized violence game (with laws of war as constraints, winner takes territory)
- **Nuclear MAD** = Deterrence game (both sides can annihilate, so both refrain)

**This document**: Shows how great powers negotiated from **"total war with nuclear risk"** (unacceptable losses) to **"non-lethal warfare"** (acceptable losses). This is ONE application of MoWD—a diplomatic effort to find a game where conflicts resolve without species extinction.

**The principle**: When current game has unacceptable losses (nuclear holocaust), negotiate DIFFERENT game with acceptable losses (non-lethal combat).
