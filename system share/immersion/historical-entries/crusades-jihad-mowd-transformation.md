# The Modern Crusades-Jihad MoWD Transformation: Triadization and Non-Lethal Conquest

## Preface: MoWD Context

**MoWD (Metagame of World Dynamics)** = The recognition that ALL political systems are games.

**What MoWD reveals**:
- **Liberal Democracy** = Vote + Capital game (majority + wealth wins)
- **Demographic competition** = Birth rate game (higher fertility dominates territory over generations)
- **Cultural assimilation** = Social pressure game (dominant culture absorbs minorities)
- **Traditional warfare** = Violence game (stronger military takes territory)

**The pattern**: Every system states "What is One/Equal" and allows "Might is Right" in the rest.

**This document**: Shows how Europe negotiated from **"demographic replacement + cultural conflict"** (unacceptable: cultural extinction) to **"territorial separation + non-lethal competition"** (acceptable: each civilization gets space). This is ONE application of MoWD—finding a game where civilizations can compete without "Rivers of Blood."

**The principle**: When current game threatens extinction (demographic replacement), negotiate DIFFERENT game with acceptable losses (territorial competition, not cultural genocide).
