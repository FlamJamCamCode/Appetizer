Horizontal Scroll Navigation - Default Mode

Auto-detects sections with IDs. Scroll down to see the navigation bar sync with your position.

Introduction

This is the first section. The horizontal navigation bar above will show a button for this section. As you scroll through this section, the button will fill with a gradient showing your progress.

The viewport indicator (blue/purple gradient) shows where your current viewport is within the section, normalized to the button width. This allows you to see your position even in very long sections.

Scroll down to see how the navigation bar moves horizontally to keep the active section's button visible, and how buttons fill as you progress through each section.

Features

This section demonstrates the key features of the horizontal scroll navigation:

  • Progress Fill: Each button fills with a red/gold gradient based on how far you've scrolled through the section
  • Viewport Indicator: A blue/purple gradient shows your current viewport position within the section
  • Normalized Scrolling: The horizontal nav bar scrolls proportionally to your vertical scroll, regardless of section size
  • Margin Handling: Smooth transitions through margins between sections without flickering
  • Click Navigation: Click any button to smoothly scroll to that section

Usage

To use this plugin, simply include the script and it will auto-detect sections with IDs: 

<script src="../lib/horizontal-scroll-nav.js"></script>

The plugin will automatically:

  • Find all sections with IDs
  • Create a horizontal navigation bar
  • Generate buttons for each section
  • Sync with your scroll position

Customization

You can customize the plugin by passing options: 

new HorizontalScrollNav({
    sectionSelector: '.my-sections',
    marginBottom: 50,
    getSectionLabel: (section, index) => {
        return section.dataset.title || `Section ${index + 1}`;
    }
});

Conclusion

This is the final section. Notice how the navigation bar has been scrolling horizontally as you moved through each section. The buttons fill based on your progress, and the viewport indicator shows exactly where you are.

Try clicking on a button in the navigation bar to jump to that section. The scroll will be smooth, and the navigation will update correctly once you arrive.

This plugin makes it easy to add this cool navigation feature to any page with sections!
